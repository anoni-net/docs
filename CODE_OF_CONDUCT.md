# 行為準則 | Code of Conduct

這份行為準則摘自 anoni.net 社群[治理章程](https://anoni.net/docs/zh-TW/community/governance/)的「行為準則」與「違反處置」兩節，獨立放在這裡，方便第一次接觸的人快速看到。完整的決策方式與爭議處理流程以治理章程為準。

## 適用範圍

本準則適用於 anoni.net 的協作與互動場域：

- 文件站（zh-TW、zh-CN、en）與相關 GitHub repo 的 PR、Issue、Discussion
- 自架的 Matrix、Cryptpad、Etherpad、Send、SearXNG、Formbricks
- 以 anoni.net 名義舉辦的工作坊、小聚、研討會
- 與 EFF、Tor Project、OONI 等國際組織合作時的對外代表

不涵蓋成員以個人身分在外部平台的言行，但在 anoni.net 場域內的言行需符合本準則。

## 互動原則

- 互相尊重：不同背景、技術熟悉度、政治立場的成員一視同仁。
- 對事不對人：批評想法，不批評身分。
- 積極預設：先假設對方的提議出自善意，再討論細節。
- 資訊透明：相關決策盡量在公開 room 進行，私下協調事後同步到公開。
- 新人友善：對基本問題保持耐心，把複雜的脈絡拆解清楚。

## 合法前提

所有協作與討論以合法用途為前提。社群不協助也不指導下列行為：

- 洗錢、規避稅務、規避經濟制裁
- 騷擾、跟蹤、人身威脅、未授權入侵
- 散布兒少性剝削內容、人口販運、毒品交易
- 在他國對該國公民進行情報蒐集或干預內政

匿名與隱私工具會被各種人使用，社群推廣的對象是新聞媒體、公民團體、研究者、需要保護自己的個人。發現協作對象有上述違法意圖，協作會立即終止。

## 機敏資訊

- 涉及個人資料、未公開研究、敏感受害者身分的內容，走[上傳機敏資訊流程](https://anoni.net/docs/zh-TW/community/upload-sensitive/)。
- 不在公開 room 直接貼受害者個資或未公開的爆料原文。
- 引用敏感報導時保留原作者來源，不擅自摘要敏感細節。

## 通報與處置

遇到違反本準則的情況，可向核心成員回報，或來信 <whisper@anoni.net>。寄到這個信箱的訊息由核心成員輪值處理。

處置由核心成員依個案決定，由輕至重：

1. 私下提醒
2. 公開警告（在相關 Matrix room 記錄）
3. 暫時隔離（暫停 Matrix、Cryptpad、GitHub 權限數天到數週）
4. 永久移除（撤銷所有自架服務帳號與參與資格）

嚴重事件（騷擾、跟蹤、人身威脅）會立即移除違規成員的存取權限，事後在公開 room 簡述處置，不揭露受害者身分，並依需要協助受害者報警或聯繫援助資源。涉及核心成員自身的爭議，由其他核心成員處理，必要時邀請外部仲裁。

## 修訂

本準則隨治理章程一起維護。修訂提案在 GitHub 開 Issue、同步到 Matrix 公開 room，至少 7 天討論期，達共識後合併。

---

**English (short):** This Code of Conduct is excerpted from the anoni.net community [governance charter](https://anoni.net/docs/zh-TW/community/governance/). It applies to all anoni.net collaboration spaces: the docs repositories, self-hosted services, events, and external representation. Be respectful, critique ideas rather than people, and assume good faith. All collaboration must be for lawful purposes. Report violations to <whisper@anoni.net>; core members handle enforcement, ranging from a private reminder to permanent removal.
