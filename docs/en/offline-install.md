# :material-download: Install this site as an offline app

When the network is cut off or blocked, you can't reach the page that tells you what to do. While you still have a connection, install this documentation site as an app (PWA, a website installed onto your device like a regular app). The core chapters download to your device, so you can still read them with no connection.

!!! tip "Do this now"
    This is advance preparation. Installing after the network is already down is too late. One minute now keeps an offline copy of the guide on your phone or computer.

How to install (open [anoni.net/docs](index.md) in a regular browser, not Tor Browser yet, see why below):

- **Android (Chrome, Edge, etc.)**: open the site, tap the browser menu, and choose "Install app" or "Add to Home screen".
- **Desktop (Chrome, Edge)**: an install icon appears at the right end of the address bar, or choose "Install anoni.net Docs" from the menu.
- **iPhone, iPad (Safari)**: tap "Share", then "Add to Home Screen".

Offline behaviour after installing:

- The core chapters under Concepts and Scenarios download at install time and stay readable offline.
- Pages you visited before remain on your device and open as usual.
- Pages that were never cached show a "You are offline" placeholder that points you back to saved content. Reload once the connection recovers to return to the latest version.

Note: Tor Browser and the onion and IPFS builds don't offer offline install, for privacy reasons (these builds don't register a background Service Worker). When you need an offline fallback, install from anoni.net in a regular browser, and go back to Tor Browser for everyday anonymous browsing.
