# Tails 7.6: bridges you can actually find, and a quieter password manager swap

![Tails](./assets/images/tails.png){style="border-radius: 10px;"}

We follow [Tails releases](https://tails.net/news/version_7.6/){target="_blank"} because they ship the same building blocks many of us recommend in real life: Tor, a hardened desktop, and tools for people who cannot assume a “normal” network path. **7.6**, dated **2026-03-26**, is worth translating not for one killer feature, but for two changes that affect **how people get online** and **how they store secrets** on a live system.

## Why this one matters (especially in regional context)

**Tor bridges** are not exotic; they are often the difference between “Tor works” and “Tor never connects.” In places where **Tor traffic is filtered or throttled**, users learn to hunt for bridges through side channels—paste sites, trusted contacts, or ad‑hoc instructions. Tails 7.6 brings that guidance **into the Tor Connection assistant**: pick **Connect to Tor automatically**, and if the network blocks Tor outright, the bridge screen can **Ask for a Tor bridge based on your region**, pulling candidates via the Tor Project’s **[Moat](https://gitlab.torproject.org/tpo/anti-censorship/rdsys/-/blob/main/doc/moat.md){target="_blank"}** service—the same family of tech Tor Browser has used since **[11.5](https://blog.torproject.org/new-release-tor-browser-115/){target="_blank"}**—with the fetch **disguised** using **[domain fronting](https://en.wikipedia.org/wiki/domain%20fronting){target="_blank"}**.

For readers in **Taiwan** and across **East/Southeast Asia**: censorship models differ, but the **pattern** is familiar—TLS interception, routing games, or “soft” blocking that fails open only for some apps. A Tails image that **surfaces bridge acquisition in-product** lowers the bar for journalists, lawyers, and civil‑society volunteers who already juggle operational risk; they should not also have to memorize bridge workflows from blog posts.

The second headline is **[Secrets](https://gitlab.gnome.org/World/secrets){target="_blank"}** replacing **[KeePassXC](https://keepassxc.org/){target="_blank"}**. That is a **product decision**, not a security downgrade by default: Secrets is tighter with **GNOME**, which matters on Tails because **accessibility regressions** (on‑screen keyboard, cursor sizing) are real blockers for some users. KeePassXC power users can still add it via **[Additional Software](https://tails.net/doc/persistent_storage/additional_software/index.en.html){target="_blank"}**; the **database format overlaps**, so migration is meant to be frictionless.

## Technical changes (short)

**New / UX**

- Tor Connection: **region‑aware bridge suggestions** via Moat; automatic path aligns with modern Tor Browser connection UX.
- Passwords: **Secrets** as default; KeePassXC optional; shortcut layout is KeePassXC‑like with extra **Shift** modifiers—see upstream docs for the full map (**Ctrl+?** in Secrets).

**Versions**

- **Electrum** → [4.7.0](https://github.com/spesmilo/electrum/blob/master/RELEASE-NOTES){target="_blank"} (from 4.5.8).
- **Tor Browser** → [15.0.8](https://blog.torproject.org/new-release-tor-browser-1508/){target="_blank"}.
- **Thunderbird** → [140.8.0](https://www.thunderbird.net/en-US/thunderbird/140.8.0esr/releasenotes/){target="_blank"}.
- **Firmware** bundles refreshed for newer GPUs, Wi‑Fi, etc.

**Fixes**

- Translated confirmation dialog before persisting **language/keyboard** to the stick ([#21448](https://gitlab.tails.boum.org/tails/tails/-/issues/21448){target="_blank"}).
- Thunderbird migration “Learn more” button ([#21455](https://gitlab.tails.boum.org/tails/tails/-/issues/21455){target="_blank"}).
- Turkish automatic upgrades ([#21466](https://gitlab.tails.boum.org/tails/tails/-/issues/21466){target="_blank"}).

Full upstream detail: [debian/changelog](https://gitlab.tails.boum.org/tails/tails/-/blob/master/debian/changelog){target="_blank"}.

## Getting 7.6

**Automatic upgrades** from **Tails 7.0+**. If that fails, use the [manual upgrade](https://tails.net/doc/upgrade/index.en.html#manual){target="_blank"} path. Clean installs: [Windows](https://tails.net/install/windows/index.en.html){target="_blank"}, [macOS](https://tails.net/install/mac/index.en.html){target="_blank"}, [Linux](https://tails.net/install/linux/index.en.html){target="_blank"}, [expert](https://tails.net/install/expert/index.en.html){target="_blank"}. Reinstalling **wipes** Persistent Storage on that medium. Image‑only downloads: [USB](https://tails.net/install/download/index.en.html){target="_blank"}, [ISO](https://tails.net/install/download-iso/index.en.html){target="_blank"}.
