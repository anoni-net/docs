# :material-wifi-off: 当前离线中

你的设备目前没有网络连接，而这个页面还没有被缓存下来。

别担心，之前浏览过的页面已经存在设备里，回到上一页或从首页开始，仍然可以离线阅读。指南的核心章节（概念、工具、场景）在安装时就已预先下载。

## 可以试试

- 回到[首页](index.md)浏览已缓存的内容
- 检查 Wi-Fi 或移动网络是否正常
- 如果你身处网络被封锁或干扰的环境，恢复连接后可以参考[翻墙与审查规避工具](tools/index.md)

连接恢复后刷新此页，就会回到正常内容。
