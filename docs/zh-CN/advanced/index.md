# :material-lock-outline: 进阶层

读完概念层与工具层之后，这个分类进到几个技术细节较深的主题。内容偏向原理与选型，适合想了解「为什么这个工具是这样设计」、「下一个十年的演化方向」的读者。涵盖的主题包含端对端加密的数学与工程、后量子密码学的转换时程，以及去中心化网站发布（IPFS、Onion）在实务上的取舍。每篇文章会尽量保留严谨度，但不堆叠术语，搭配范例与图示说明。

## 文章一览

- [端对端加密如何运作](./e2ee.md)：密钥交换、前向保密、群组消息的工程取舍。
- [后量子密码概观](./post-quantum.md)：NIST 标准的进度与真实系统的转换时程。
- [去中心化网站发布](./dweb-ipfs-onion.md)：IPFS 与 Onion 的差别与合用场景。
- [零知识身分验证与支付](./zk-identity-payments.md)：环签名、zk-SNARKs、链上分析的能与不能。
