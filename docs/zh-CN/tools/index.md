# :material-toolbox-outline: 工具层

读完[概念层](../basics/index.md)后，这个分类介绍几个在匿名网络讨论中最常被提到的工具：Tor 提供匿名连线，Tails 是即用即丢的操作系统，OONI 观测网络审查的实际样态。我们也补上几篇进阶主题，例如 Tor Browser 的进阶设定、不同匿名操作系统的选择、即时通讯工具的比较，以及密码管理器入门。每篇文章都搭配[威胁模型](../basics/threat-model.md)说明在什么情境下适合或不适合使用，避免「工具当答案」的误区。
