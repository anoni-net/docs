# 严选报告

我们会找寻与相关主题的报告或论文，尝试翻译成中文后再加入一些注解、解释呈现。

!!! info "报告中译版仅在 zh-TW 维护"

    严选报告类内容的中译版本目前只在**正体中文（zh-TW）**版本维护，简体中文版暂不另做一份翻译。建议海外华语读者直接到 [zh-TW 严选报告](https://anoni.net/docs/zh-tw/reports/){target="_blank"} 阅读完整内容。

    长文报告的简繁差异多半在地名、技术术语用词，不影响阅读。如果未来社群人力允许，会考虑补 zh-CN 版本，但不在 2026 路线图内。
