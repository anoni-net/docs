# :material-file-lock: 上传机敏信息流程

为了确保重要个人信息在传递过程中，不被非必要人员存取、阅读，且传递媒介可安全的遗忘与删除，请透过以下方式将个人资料、机敏资料上传与回复。

## 上传流程

1. 前往 [Send](https://send.anoni.net/){target="_blank"}（为匿名网络社群自架服务之一）。
2. 选入要上传的文件。
3. 调整失效选项：「**一次下载**」、「**7 天**」后失效。
4. 勾选「**:octicons-check-circle-24:{ style="color: green;" } 使用密码保护**」，填入**约定的**密码保护。
5. 确认无误后「**上传**」。

上传后会产生一组分享链接，请「复制链接」后回复给收件人，完成个人资料、机敏资料回传。

!!! info ""

    [Send](https://github.com/timvisee/send){target="_blank"} 是一个简易的文件共享自架软件，透过端到端加密的方式来分享文件，并提供会自动失效的链接，这样的设计可以保留分享时的隐私，也确保文件不会永久保存在网络上、邮件服务中。
