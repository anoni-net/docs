# :material-account-group: 社群参与

我们以「**匿名网络**」为核心，透过 Tor、Tails、OONI 等工具在台湾推广网络自由与隐私。2026 年聚焦三大主题，欢迎有兴趣的伙伴选择一个主题加入讨论与协作。我们使用自架 Matrix 家服务器与申请制账号，在隐私与社群品质之间取得平衡，背后考量可参见下方延伸阅读。

## 2026 三大主题

每个主题都有独立的研究专题入口页，整理研究目标、相关文章索引、进度与如何加入。

- [**个人隐私指引**](./privacy-guide.md)：整理可实际操作的隐私保护指引，依情境（日常、敏感工作、高风险）提供工具与步骤。
- [**Tor Relay 校园建立**](./relay-on-campus.md)：与 EFF、Tor Project 合作推动校园中继节点，已有台师大成功案例，目标整理申请流程与活动企划。
- [**匿名支付**](./payments-research.md)：探索现金以外情境下的匿名支付（稳定币、区块链应用），含法规与实作面向。

完整 2025 回顾与 2026 起步背景见：[延续 2025，走向 2026](../blog/posts/2025to2026.md)。
