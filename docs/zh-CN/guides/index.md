# :material-compass-outline: 指南

「指南」整合了 anoni.net 的核心阅读路径，从基础概念走到具体工具、实际应用场景、进阶主题，再到对外的研究报告，五个分类涵盖从理论到实践再到深度延伸。每个层次有自己的 index 页列出所有文章。

## 五个阅读层次

<div class="grid cards" markdown>

- [:material-school-outline: 概念](../basics/index.md)

    匿名与隐私的核心思考工具：词汇厘清、威胁模型、Metadata、网络自由、匿名支付。从这里建立共同的动机与词汇。

- [:material-toolbox-outline: 工具](../tools/index.md)

    具体工具的介绍与比较：Tor、Tails、OONI、Tor Browser、消息工具、密码管理器、加密货币隐私光谱。

- [:material-account-group-outline: 场景](../scenarios/index.md)

    特定角色或情境的应用：记者、社运参与者、家暴幸存者、LGBTQ+ 与性少数、选举观察员、匿名捐款。

- [:material-lock-outline: 进阶](../advanced/index.md)

    技术深度的延伸阅读：端对端加密、后量子密码、去中心化发布、零知识身分与支付。

- [:material-file-pdf-box: 报告](../reports/index.md)

    外部研究的中译与整理：包含 InterSecLab 的「网络政变」报告。

</div>

## 如何走这条路径

不必照顺序读完全部 5 个分类，几种切入方式：

- **第一次接触匿名网络**：从 [概念](../basics/index.md) 开始，依序读完 5 篇建立词汇与动机
- **想找特定工具**：直接到 [工具](../tools/index.md) 找对应工具的介绍与威胁模型
- **有具体需求**：到 [场景](../scenarios/index.md) 找最接近自己的角色，看完整流程
- **想理解技术原理**：到 [进阶](../advanced/index.md) 看 E2EE、后量子等深度主题
- **看外部资源**：到 [报告](../reports/index.md) 看翻译整理过的研究

延伸到在地的法规与观测，可以接著看 [在地脉络](../taiwan/index.md)。想加入社群协作，见 [社群参与](../community/index.md)。
