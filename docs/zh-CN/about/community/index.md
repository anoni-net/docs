# :material-account-group: 社区参与

我们以「**匿名网络**」为核心，通过 Tor、Tails、OONI 等工具在台湾推广网络自由与隐私。2026 年聚焦三大主题，欢迎有兴趣的伙伴选择一个主题加入讨论与协作。

## 2026 三大主题

- **个人隐私指引**：整理可实际操作的隐私保护指引，依情境（日常、敏感工作、高风险）提供工具与步骤。
- **Tor Relay 校园建设**：与 EFF、Tor 合作推动校园中继节点，已有师大成功案例，目标是整理申请流程与活动规划。
- **匿名支付**：探索现金以外情境下的匿名支付（如稳定币、区块链应用），包含法规与实践面向。

详细说明与规划请见：[延续 2025，走向 2026：个人隐私指引、Tor Relay 校园建设竞赛、匿名支付探索](../../blog/posts/2025to2026.md)。
