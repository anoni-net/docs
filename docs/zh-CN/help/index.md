# :material-lifebuoy: 紧急求救

如果你正在经历账号被盗、装置遗失、跟踪骚扰、家庭暴力、突然被断网或被审查，这页提供一份「现在可以做什么」的清单，按情境分类，尽量让你在压力下能直接照著步骤走。

这页不是法律意见，也不取代专业协助。情境急迫时请优先联系下方列出的求助专线。

!!! note "给海外华语读者"

    以下流程含**台湾在地求助管道**（110、113、165 等专线，与台湾保护组织、政府机关）。海外华语读者可参照思路，但求助专线请依当地对应资源替换。Access Now 数位安全求助热线对全球记者、人权工作者、社运参与者开放，多语支援。

## 24 小时可用的台湾求助专线

拨打不需要费用，多数提供匿名咨询。

- **113 保护专线**：家庭暴力、性侵害、儿少保护的全国通报专线（卫生福利部）。手机、市话皆可拨。
- **110 警察报案**：人身安全立即受威胁、犯罪正在发生时拨打。
- **165 反诈骗咨询**：账号被盗后对方要求转账、收到可疑消息、怀疑是诈骗时拨打。
- **1995 安心专线**：心理支持、自杀防治、情绪陪伴（卫生福利部）。
- **iWIN 网络内容防护机构**：18 岁以下涉及网络侵害内容的通报窗口，网站 [https://i.win.org.tw/](https://i.win.org.tw/){target="_blank"}。

## 账号被盗或被异常登录

### 第一时间（5 分钟内）

1. 从你信任的另一台装置登录该服务，看是否还能进入
2. 立即更换密码，并把新密码存进密码管理器（不要用浏览器记住）
3. 开启两步验证（2FA）。如果原本就有开且验证码也被换掉，走服务的「账号回复」流程
4. 登出所有其他装置（多数服务在「安全性」或「装置」页有此选项）

### 接下来的 30 分钟

- 检查近期登录记录、寄件备份匣、第三方授权应用：是否有不明 IP、不明寄出邮件、不熟悉的授权
- 撤销所有可疑的第三方应用授权
- 通知近期可能被冒用联系的亲友：注意社交工程诈骗
- 如果被盗账号是 Email：高优先处理。Email 通常是其他账号的回复管道，要先稳住

### 之后的整理

- 把所有关键账号（Email、银行、社群、云端）的密码换成独立、随机的长密码
- 全部开启 2FA，优先使用 TOTP（Authy、Aegis 等）或硬件密钥，尽量不用短信
- 进一步阅读：[密码管理器入门](../tools/password-manager.md)、[威胁模型如何建立](../basics/threat-model.md)

## 装置遗失或被偷

### 立刻做

1. 从另一台装置登录装置制造商或操作系统的「寻找我的装置」服务（Apple Find My、Google Find Hub、Microsoft、Samsung 对应页），标记为遗失，启动远端锁定
2. 拨打 165 确认没有相关诈骗风险
3. 通报电信业者挂失 SIM 卡（短信验证码是常见冒用入口）
4. 变更该装置上所有已登录服务的密码（从另一台装置）

### 报案

- 拨打 110 或前往就近派出所报案，取得受理证明（后续向业者申请补发或保险可能需要）
- 提供装置序号（IMEI、序号）给警方与业者，事先存在 Email 或实体文件可省时间

### 后续

- 检查云端备份是否被异常下载
- 对曾经仅在该装置储存的敏感资料（密码、私钥、护照扫描档）做风险评估
- 进一步阅读：[什么是 Tails](../tools/what-is-tails.md)（即用即丢操作系统，敏感工作的替代方案）

## 跟踪骚扰与线上骚扰

### 安全评估

- 你的住处、职场、上下学路线，骚扰者是否知情或能查到？
- 骚扰是否扩及亲友、雇主、同事？
- 是否有具体的恐吓、威胁或不请自来的接近？

### 求助管道

- **拨打 113 保护专线**：咨询与评估，必要时转介社工
- **保护令申请**：透过警察、社政或地方法院家事庭申请。紧急保护令可在数小时内核发
- **跟踪骚扰防制法**：自 2022 年施行，对不受欢迎的接近、跟监、拨打骚扰电话、寄送物品等都有刑事追诉依据

### 在地支援组织

- **现代妇女基金会**：[https://www.38.org.tw/](https://www.38.org.tw/){target="_blank"}（性别暴力、跟踪骚扰的咨询与服务）
- **妇女救援基金会**：[https://www.twrf.org.tw/](https://www.twrf.org.tw/){target="_blank"}
- **励馨基金会**：[https://www.goh.org.tw/](https://www.goh.org.tw/){target="_blank"}
- 各县市政府社会局家庭暴力暨性侵害防治中心

### 数位防护的同步动作

- 把社群媒体切换到仅限好友或私人账号
- 移除照片中的位置标签、Metadata。新照片上传前以 Tails 内建的 Metadata Cleaner 或类似工具去识别
- 关闭相关 App 的位置共享（Google Maps、Find My）
- 进一步阅读：[家暴受害者的数位准备](../scenarios/domestic-violence.md)、[Metadata 是什么](../basics/metadata.md)

## 家庭暴力或亲密关系暴力

人身安全永远优先于数位安全。如果情况急迫，请立即拨打 110 或 113。

下方列的是当事人「还没准备离开」或「离开过程中」的数位面准备：

- 共用装置上的搜寻记录、地图记录、通讯记录可能会被检视，准备一台对方不知情的装置作为求助管道（公共图书馆电脑、信任亲友的装置都可以）
- 共用家庭方案的云端、定位（Find My、Google Family）可能让对方看到你的位置，先确认再决定要不要离开
- 加害者偶尔会在当事人装置上偷装追踪类 App。如果离开后仍感觉位置被掌握，请就近警察局或保护专线协助检测
- 详细流程请见 [家暴幸存者的数位准备](../scenarios/domestic-violence.md)（撰写中）

## 被断网、被审查或被封锁

情境差异大，列出常见的应对方向：

- **特定网站打不开**：先用 OONI Probe 或 [OONI Explorer](https://explorer.ooni.org/){target="_blank"} 确认是个别封锁还是区域问题，可参考 [ASNs 自治网络观测资料分析](../taiwan/ooni-asn-coverage.md) 看 TW 的观测现况
- **整个家用网络被中断**：先确认是否电信端的问题（拨打 ISP 客服）。如果怀疑是针对性处置，记录时间、IP、服务名称与消息截图
- **跨境连线受限**：尝试 Tor Browser 与内建桥接（obfs4、Snowflake、meek-azure）。桥接点建立可参考 [Tor Snowflake](../tools/tor-snowflake.md)
- **完全无法连上 Tor**：透过电子邮件（[bridges@torproject.org](mailto:bridges@torproject.org)）申请非公开桥接

## 趁有网络时，把文档站装成离线 App

断网或被封锁时，连不上网就查不到这页的应对步骤。趁现在还能连接，把这个文档站装成 App（PWA，把网站像一般 App 一样装到设备上），核心章节会先下载到设备，之后没网络也能查阅。

!!! tip "建议现在就做"
    这是预先准备。等网络断了才装就来不及，花一分钟装好，等于在手机或电脑里先存一份离线版指南。

安装方式（用一般浏览器打开 [anoni.net/docs](../index.md)，先别用 Tor Browser，原因见下）：

- **Android（Chrome、Edge 等）**：打开站点后点浏览器菜单，选「安装应用」或「添加到主屏幕」。
- **电脑（Chrome、Edge）**：地址栏右侧会出现安装图标，或从菜单选「安装 anoni.net Docs」。
- **iPhone、iPad（Safari）**：点「分享」按钮，再选「添加到主屏幕」。

装好后的离线行为：

- 概念、工具、场景的核心章节在安装时就先下载，断网也能读。
- 之前浏览过的页面会留在设备上，照常打开。
- 没缓存到的页面断网时会显示「当前离线中」提示页，引导你回到已存的内容。连接恢复后刷新，就会回到最新版本。

要注意：Tor Browser 以及 onion、IPFS 版基于隐私考量不提供离线安装（这些版本不会注册后台的 Service Worker）。需要离线备援时用一般浏览器在 anoni.net 安装，平时的匿名浏览再回到 Tor Browser。

## 个资外泄或被公开（Doxxing）

### 立即动作

1. 截图保留证据（含 URL、发布时间、发布者）
2. 对该平台提出检举，附上证据与「侵害个人资料」的明确主张
3. 对特定平台不应的情况下，可向 [iWIN](https://i.win.org.tw/){target="_blank"}（儿少相关）或 [个人资料保护委员会](https://www.pdpc.gov.tw/){target="_blank"} 提出协助申请
4. 咨询专业律师考虑民事或刑事追诉

### 同步保护

- 变更可能受牵连的账号密码
- 通知关键联络人，提醒对方注意可能的社交工程
- 进一步阅读：[个资法 2025 修法](../taiwan/pdpa-2025.md)

## 旅居或跨境工作的常见资源

- **外交部紧急联络中心**：海外台湾国民人身安全急难救助，国内 0800-085-095，国外拨打 +886-800-085-095
- 各驻外馆处与当地警政管道，依出国前查询
- Access Now Digital Security Helpline：[https://www.accessnow.org/help/](https://www.accessnow.org/help/){target="_blank"}（给记者、人权工作者、社运参与者的数位安全求助，多语支援）

## 事件结束后的整理

事情缓下来后值得做的：

- 把这次的处置流程写下来，留作下次或别人的参考（不需公开，可以是私人笔记）
- 评估这次曝光的范围：哪些密码还没换、哪些账号还没查
- 重新走一遍 [威胁模型](../basics/threat-model.md)：这次经验有没有改变你对「保护什么、防谁、能付出多少成本」的判断
- 必要时寻求心理支持（1995 安心专线或专业咨商）

## 我们不能做的事

匿名网络社群是技术社群，不是法律服务、不是心理咨商、不是执法单位。我们能：

- 整理工具与资源让你判断
- 在数位防护面提供经验
- 把你转介到适合的专业组织

我们不能：代为提告、发出保护令、进入装置取证、提供法律意见。情境急迫请优先联系专业专线。
