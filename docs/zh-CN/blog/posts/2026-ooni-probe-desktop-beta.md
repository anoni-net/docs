# OONI 新版 Probe Desktop 测试邀请

![OONI](./assets/images/ooni-run-v2.webp){style="border-radius: 10px;"}

[OONI](https://ooni.org/){target="_blank"} 团队推出了新的跨平台 **OONI Probe Desktop** 应用程序，并内置全新的仪表盘界面。团队希望邀请大家协助测试。

你可以在这里下载新版的**内部测试版**：[OONI Probe Desktop v6.0.1（GitHub Releases）](https://github.com/ooni/probe-multiplatform/releases/v6.0.1){target="_blank"}

根据你的测试结果，可以用下列方式向团队提供意见：

1. 若在程序中遇到问题，请通过应用程序提交**错误报告**。这样团队才能一并取得应用程序记录，方便排查。
2. 若有其他**不像是单一程序错误**的反馈，可写信至 [team@ooni.org](mailto:team@ooni.org)。
3. 若你使用 GitHub，也可在仓库中开 issue：[ooni/probe-multiplatform/issues](https://github.com/ooni/probe-multiplatform/issues){target="_blank"}

为了协助团队排查问题，反馈时请一并附上下列信息（可直接复制后填写）：

- **PLATFORM**（例如 macOS、macOS Sonoma、Tahoe、Windows 10、Windows 11 等）：
- **MODEL**（例如 MacBook Pro、MacBook Air、iMac、Mac mini、Mac Studio 等）：
- **OS Version**（例如 12.2.1、13.5、14.3.1、22H2、23H2、21H2 等）：

感谢你的协助与反馈。

～ OONI 团队
