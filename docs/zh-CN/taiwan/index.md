# :material-island: 在地脉络

!!! note "给海外华语读者的导读"

    以下文章记录台湾的法规、观测、研究案例，作为一个民主社会在隐私、监管、加密之间取舍的参考。对其他华语圈读者是参照组，并非直接套用的指引。法规面向（个资法、VASP、揭弊者保护）每篇文末附「中国对应制度速览」连到外部资源，但本节不重写成中国脉络的主题文，那需要另一个研究路径。

匿名与网络自由的讨论放在台湾，会有不同的问题意识。这个分类沿用 [网络自由为什么重要](../basics/internet-freedom.md) 提到的三个面向：连线层、个资与身分、金流，把每个面向在台湾的观测资料、法规动态与制度讨论整理在一起。内容适合给研究者、记者、倡议组织，或对在地政策有兴趣的读者，当作切入观察的起点。

## 文章一览

### 连线层的在地观测

对应 anoni.net 长期维运的观测资料工作，看「台湾的连线环境实际长怎样」。

- [OONI 网站检测清单](./ooni-checklist.md)：在地清单如何维护、分类与更新，是观测资料的入门起点。
- [ASN 自治网络观测资料分析](./ooni-asn-coverage.md)：台湾各 ASN 的 OONI 测试覆盖与盲点。
- [Tor Relays 观测点](./tor-relay-watcher.md)：台湾 Tor 中继节点的即时运作状况与区域比较。

### 个资与身分

法规面决定了个人对自己资料的掌握程度。

- [台湾个资法 2025 修法](./pdpa-2025.md)：个保会的设立、罚则与跨境传输规范调整。

### 金流

金流匿名是 2026 三大主题之一，台湾的加密资产监理正在从登记制走向许可制。

- [台湾 VASP 法 2026](./vasp-2026.md)：行政院《虚拟资产服务法》草案的架构、刑责与过渡期。

### 制度与技术交界

法律保护的范围与技术匿名能力之间，存在一段需要讨论的距离。

- [揭弊者保护法的技术观察](./whistleblower-law.md)：2025 年上路的《公益揭弊者保护法》保护什么、私部门缺口在哪、揭弊者本人如何准备。
