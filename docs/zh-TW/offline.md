# :material-wifi-off: 目前離線中

你的裝置目前沒有網路連線，而這個頁面還沒有被快取下來。

別擔心，之前瀏覽過的頁面已經存在裝置裡，回到上一頁或從首頁開始，仍然可以離線閱讀。指南的核心章節（概念、工具、場景）在安裝時就已預先下載。

## 可以試試

- 回到[首頁](index.md)瀏覽已快取的內容
- 檢查 Wi-Fi 或行動網路是否正常
- 如果你身處網路被封鎖或干擾的環境，恢復連線後可以參考[翻牆與審查規避工具](tools/index.md)

連線恢復後重新整理此頁，就會回到正常內容。
