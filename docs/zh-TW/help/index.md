# :material-lifebuoy: 緊急求救

如果你正在經歷帳號被盜、裝置遺失、跟蹤騷擾、家庭暴力、突然被斷網或被審查，這頁提供一份「現在可以做什麼」的清單，按情境分類，盡量讓你在壓力下能直接照著步驟走。

這頁不是法律意見，也不取代專業協助。情境急迫時請優先聯繫下方列出的求助專線。

## 24 小時可用的台灣求助專線

撥打不需要費用，多數提供匿名諮詢。

- **113 保護專線**：家庭暴力、性侵害、兒少保護的全國通報專線（衛生福利部）。手機、市話皆可撥。
- **110 警察報案**：人身安全立即受威脅、犯罪正在發生時撥打。
- **165 反詐騙諮詢**：帳號被盜後對方要求轉帳、收到可疑訊息、懷疑是詐騙時撥打。
- **1995 安心專線**：心理支持、自殺防治、情緒陪伴（衛生福利部）。
- **iWIN 網路內容防護機構**：18 歲以下涉及網路侵害內容的通報窗口，網站 [https://i.win.org.tw/](https://i.win.org.tw/){target="_blank"}。

## 帳號被盜或被異常登入

### 第一時間（5 分鐘內）

1. 從你信任的另一台裝置登入該服務，看是否還能進入
2. 立即更換密碼，並把新密碼存進密碼管理器（不要用瀏覽器記住）
3. 開啟兩步驟驗證（2FA）。如果原本就有開且驗證碼也被換掉，走服務的「帳號回復」流程
4. 登出所有其他裝置（多數服務在「安全性」或「裝置」頁有此選項）

### 接下來的 30 分鐘

- 檢查近期登入紀錄、寄件備份匣、第三方授權應用：是否有不明 IP、不明寄出信件、不熟悉的授權
- 撤銷所有可疑的第三方應用授權
- 通知近期可能被冒用聯繫的親友：注意社交工程詐騙
- 如果被盜帳號是 Email：高優先處理。Email 通常是其他帳號的回復管道，要先穩住

### 之後的整理

- 把所有關鍵帳號（Email、銀行、社群、雲端）的密碼換成獨立、隨機的長密碼
- 全部開啟 2FA，優先使用 TOTP（Authy、Aegis 等）或硬體金鑰，盡量不用簡訊
- 進一步閱讀：[密碼管理器入門](../tools/password-manager.md)、[威脅模型如何建立](../basics/threat-model.md)

## 裝置遺失或被偷

### 立刻做

1. 從另一台裝置登入裝置製造商或作業系統的「尋找我的裝置」服務（Apple Find My、Google Find Hub、Microsoft、Samsung 對應頁），標記為遺失，啟動遠端鎖定
2. 撥打 165 確認沒有相關詐騙風險
3. 通報電信業者掛失 SIM 卡（簡訊驗證碼是常見冒用入口）
4. 變更該裝置上所有已登入服務的密碼（從另一台裝置）

### 報案

- 撥打 110 或前往就近派出所報案，取得受理證明（後續向業者申請補發或保險可能需要）
- 提供裝置序號（IMEI、序號）給警方與業者，事先存在 Email 或實體文件可省時間

### 後續

- 檢查雲端備份是否被異常下載
- 對曾經僅在該裝置儲存的敏感資料（密碼、私鑰、護照掃描檔）做風險評估
- 進一步閱讀：[什麼是 Tails](../tools/what-is-tails.md)（即用即丟作業系統，敏感工作的替代方案）

## 跟蹤騷擾與線上騷擾

### 安全評估

- 你的住處、職場、上下學路線，騷擾者是否知情或能查到？
- 騷擾是否擴及親友、雇主、同事？
- 是否有具體的恐嚇、威脅或不請自來的接近？

### 求助管道

- **撥打 113 保護專線**：諮詢與評估，必要時轉介社工
- **保護令申請**：透過警察、社政或地方法院家事庭申請。緊急保護令可在數小時內核發
- **跟蹤騷擾防制法**：自 2022 年施行，對不受歡迎的接近、跟監、撥打騷擾電話、寄送物品等都有刑事追訴依據

### 在地支援組織

- **現代婦女基金會**：[https://www.38.org.tw/](https://www.38.org.tw/){target="_blank"}（性別暴力、跟蹤騷擾的諮詢與服務）
- **婦女救援基金會**：[https://www.twrf.org.tw/](https://www.twrf.org.tw/){target="_blank"}
- **勵馨基金會**：[https://www.goh.org.tw/](https://www.goh.org.tw/){target="_blank"}
- 各縣市政府社會局家庭暴力暨性侵害防治中心

### 數位防護的同步動作

- 把社群媒體切換到僅限好友或私人帳號
- 移除照片中的位置標籤、Metadata。新照片上傳前以 Tails 內建的 Metadata Cleaner 或類似工具去識別
- 關閉相關 App 的位置共享（Google Maps、Find My）
- 進一步閱讀：[家暴受害者的數位準備](../scenarios/domestic-violence.md)、[Metadata 是什麼](../basics/metadata.md)

## 家庭暴力或親密關係暴力

人身安全永遠優先於數位安全。如果情況急迫，請立即撥打 110 或 113。

下方列的是當事人「還沒準備離開」或「離開過程中」的數位面準備：

- 共用裝置上的搜尋紀錄、地圖紀錄、通訊紀錄可能會被檢視，準備一台對方不知情的裝置作為求助管道（公共圖書館電腦、信任親友的裝置都可以）
- 共用家庭方案的雲端、定位（Find My、Google Family）可能讓對方看到你的位置，先確認再決定要不要離開
- 加害者偶爾會在當事人裝置上偷裝追蹤類 App。如果離開後仍感覺位置被掌握，請就近警察局或保護專線協助檢測
- 詳細流程請見 [家暴倖存者的數位準備](../scenarios/domestic-violence.md)（撰寫中）

## 被斷網、被審查或被封鎖

情境差異大，列出常見的應對方向：

- **特定網站打不開**：先用 OONI Probe 或 [OONI Explorer](https://explorer.ooni.org/){target="_blank"} 確認是個別封鎖還是區域問題，可參考 [ASNs 自治網路觀測資料分析](../taiwan/ooni-asn-coverage.md) 看 TW 的觀測現況
- **整個家用網路被中斷**：先確認是否電信端的問題（撥打 ISP 客服）。如果懷疑是針對性處置，記錄時間、IP、服務名稱與訊息截圖
- **跨境連線受限**：嘗試 Tor Browser 與內建橋接（obfs4、Snowflake、meek-azure）。橋接點建立可參考 [Tor Snowflake](../tools/tor-snowflake.md)
- **完全無法連上 Tor**：透過電子郵件（[bridges@torproject.org](mailto:bridges@torproject.org)）申請非公開橋接

## 個資外洩或被公開（Doxxing）

### 立即動作

1. 截圖保留證據（含 URL、發布時間、發布者）
2. 對該平台提出檢舉，附上證據與「侵害個人資料」的明確主張
3. 對特定平台不應的情況下，可向 [iWIN](https://i.win.org.tw/){target="_blank"}（兒少相關）或 [個人資料保護委員會](https://www.pdpc.gov.tw/){target="_blank"} 提出協助申請
4. 諮詢專業律師考慮民事或刑事追訴

### 同步保護

- 變更可能受牽連的帳號密碼
- 通知關鍵聯絡人，提醒對方注意可能的社交工程
- 進一步閱讀：[個資法 2025 修法](../taiwan/pdpa-2025.md)

## 旅居或跨境工作的常見資源

- **外交部緊急聯絡中心**：海外台灣國民人身安全急難救助，國內 0800-085-095，國外撥打 +886-800-085-095
- 各駐外館處與當地警政管道，依出國前查詢
- Access Now Digital Security Helpline：[https://www.accessnow.org/help/](https://www.accessnow.org/help/){target="_blank"}（給記者、人權工作者、社運參與者的數位安全求助，多語支援）

## 事件結束後的整理

事情緩下來後值得做的：

- 把這次的處置流程寫下來，留作下次或別人的參考（不需公開，可以是私人筆記）
- 評估這次曝光的範圍：哪些密碼還沒換、哪些帳號還沒查
- 重新走一遍 [威脅模型](../basics/threat-model.md)：這次經驗有沒有改變你對「保護什麼、防誰、能付出多少成本」的判斷
- 必要時尋求心理支持（1995 安心專線或專業諮商）

## 我們不能做的事

匿名網路社群是技術社群，不是法律服務、不是心理諮商、不是執法單位。我們能：

- 整理工具與資源讓你判斷
- 在數位防護面提供經驗
- 把你轉介到適合的專業組織

我們不能：代為提告、發出保護令、進入裝置取證、提供法律意見。情境急迫請優先聯繫專業專線。

如果你想協助維護或補強這頁的資源，歡迎透過 [社群自架服務](../community/tools.md) 加入社群討論。
