# :material-account-voice: 揭弊者保護法的技術觀察

揭弊者保護是制度問題，但每一次揭弊行為都會留下數位足跡：通訊紀錄、檔案下載、登入時間、辦公網路日誌。法律能保護的範圍，與技術上能不能讓揭弊行為「不留痕跡到可被識別」之間，存在一段需要被仔細討論的距離。這篇文章從現行揭弊者保護相關法規切入，整理保護的對象與條件，對照技術上的匿名管道（SecureDrop、加密電郵、Tor 上的隱蔽收件箱）能補足哪些縫隙，並列出記者與律師圈在實務上的建議流程。
