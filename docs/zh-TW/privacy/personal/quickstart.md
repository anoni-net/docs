# 15 分鐘隱私健檢清單

這份清單以「現在就能做」為原則設計，先完成這幾項，再進入其他深入頁面。

## 帳號安全

- [ ] 你最常用的 Email 啟用兩步驟驗證（2FA）。
- [ ] 檢查主要帳號密碼是否重複使用，至少先更換最重要的 1-2 個帳號。
- [ ] 把系統與瀏覽器更新到最新版本。

## 裝置與鎖定

- [ ] 手機與電腦都啟用鎖定密碼或生物辨識。
- [ ] 鎖定畫面關閉不必要通知預覽。
- [ ] 設定裝置閒置自動鎖定（建議 30 秒到 2 分鐘）。

## 平台隱私設定

- [ ] 社群平台把個人資料可見範圍調成「朋友」或更小範圍。
- [ ] 關閉廣告個人化與第三方資料共享選項。
- [ ] 檢查 App 權限，先關閉不必要的定位、麥克風、相機權限。

## 瀏覽與搜尋

- [ ] 瀏覽器關閉第三方 Cookie。
- [ ] 移除不明或長期未使用的瀏覽器擴充套件。
- [ ] 把預設搜尋引擎改成你信任、較少追蹤的選項。
