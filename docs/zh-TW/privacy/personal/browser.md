# 瀏覽器與搜尋

你每天大部分的網路活動都從瀏覽器開始。這一頁提供優先順序清楚的調整建議，先從最有感、最不影響使用習慣的設定開始。

標記說明：`一般`{style="color: SeaGreen;"} 代表多數使用者可直接採用，`高風險`{style="color: OrangeRed;"} 代表較高風險情境下建議優先採用。

## 第一層：優先調整清單

1. 關閉第三方 Cookie，減少跨站追蹤（`一般`{style="color: SeaGreen;"}）。
2. 清理不必要的擴充套件，只保留你知道用途的工具（`一般`{style="color: SeaGreen;"}）。
3. 開啟「傳送不追蹤」與防追蹤選項（`一般`{style="color: SeaGreen;"}）。
4. 把預設搜尋引擎調整為你信任、較少個人化追蹤的服務（`一般`{style="color: SeaGreen;"}）。

## 第二層：逐步加強

- 使用不同瀏覽器分流不同情境，例如工作、日常、測試分開（`高風險`{style="color: OrangeRed;"}）。
- 針對高風險瀏覽需求，使用獨立視窗或臨時工作階段，降低帳號關聯（`高風險`{style="color: OrangeRed;"}）。
- 定期清除網站權限，特別是定位、通知、麥克風、相機（`一般`{style="color: SeaGreen;"}）。

## 第三層：風險意識

- 無痕模式不等於匿名，主要作用是減少本機瀏覽紀錄（`一般`{style="color: SeaGreen;"}）。
- 單一設定無法完全避免追蹤，應以多項設定組合來降低可識別性（`一般`{style="color: SeaGreen;"}/`高風險`{style="color: OrangeRed;"}）。
- 隱私設定會隨平台更新改變，建議每月檢查一次（`一般`{style="color: SeaGreen;"}）。
