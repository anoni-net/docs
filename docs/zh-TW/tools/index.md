# :material-toolbox-outline: 工具層

讀完[概念層](../basics/index.md)後，這個分類介紹幾個在匿名網路討論中最常被提到的工具：Tor 提供匿名連線，Tails 是即用即丟的作業系統，OONI 觀測網路審查的實際樣態。我們也補上幾篇進階主題，例如 Tor Browser 的進階設定、不同匿名作業系統的選擇、即時通訊工具的比較，以及密碼管理器入門。每篇文章都搭配[威脅模型](../basics/threat-model.md)說明在什麼情境下適合或不適合使用，避免「工具當答案」的誤區。
