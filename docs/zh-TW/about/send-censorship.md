# :material-file-lock: 上傳機敏資訊流程

為了確保重要個人資訊在傳遞過程中，不被非必要人員存取、閱讀，且傳遞媒介可安全的遺忘與刪除，請透過以下方式將個人資料、機敏資料上傳與回覆。

## 上傳流程

1. 前往 [Send](https://send.anoni.net/){target="_blank"}（為匿名網路社群自架服務之一）。
2. 選入要上傳的文件。
3. 調整失效選項：「**一次下載**」、「**7 天**」後失效。
4. 勾選「**:octicons-check-circle-24:{ style="color: green;" } 使用密碼保護**」，填入**約定的**密碼保護。
5. 確認無誤後「**上傳**」。

上傳後會產生一組分享連結，請「複製連結」後回覆給收件人，完成個人資料、機敏資料回傳。

!!! info ""

    [Send](https://github.com/timvisee/send){target="_blank"} 是一個簡易的檔案共享自架軟體，透過端到端加密的方式來分享檔案，並提供會自動失效的連結，這樣的設計可以保留分享時的隱私，也確保檔案不會永久保存在網路上、郵件服務中。
