# :material-account-box-outline: 關於我們

我們是關注「**匿名網路**」、「**網路自由**」的一群人，透過 Tor、Tails、OONI 工具喚起一般大眾對網路自由的重視，並在臺灣推動隱私保護的實踐。

專案起初聚焦於 OONI，透過[調查](../ooni-asns-coverage.md)臺灣 ASNs 在 OONI 資料庫中觀測資料的完整性。後來我們逐步擴展定位，進一步專注在「**匿名網路**」議題的推廣與工具建立。2025/08，我們在國立台灣科技大學順利舉辦了「[匿名網路工作坊](../event-workshop-2025.md)」，招募到第一批社群夥伴，一同展開 2026 年的工作。

**2026 年，社群聚焦三大主題：**

- **個人隱私指引**：整理可實際操作的隱私保護指引，依情境（日常、敏感工作、高風險）提供工具與步驟。
- **Tor Relay 校園建立**：與 EFF、Tor 合作推動校園中繼節點，已有台師大成功案例。
- **匿名支付**：探索現金以外情境下的匿名支付（如穩定幣、區塊鏈應用），含法規與實作面向。
