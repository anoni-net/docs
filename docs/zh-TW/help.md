# :material-lifebuoy: 緊急求救

如果你正在經歷帳號被盜、裝置遺失、被跟蹤騷擾、突然被斷網或被審查，這頁提供一份「現在可以做什麼」的清單，按情境分類，盡量讓你在壓力下能直接照著步驟走。內容包含：第一時間的優先順序、可以聯繫的求助管道（衛福部、警政署、現代婦女基金會、各 ISP 的緊急聯絡窗口）、台灣以外旅居或跨境工作者的常見資源，以及事件結束後的事後整理建議。每個情境都會標註哪些步驟需要他人協助、哪些可以自己完成。
