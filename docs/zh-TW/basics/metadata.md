# :material-file-tree: Metadata 是什麼，為什麼重要

Metadata 是「資料的資料」，例如一通電話的對象、時間、長度與位置，一封郵件的寄件人、收件人、伺服器路徑，一張照片夾帶的拍攝時間與 GPS 座標。內容雖然沒被打開，光是這些周邊資訊就足以還原一個人的社交網絡、行為模式與活動範圍。這篇文章用幾個真實案例說明 Metadata 的洩漏怎麼發生，介紹常見的 Metadata 來源（網路通訊、檔案 EXIF、系統日誌、平台後端紀錄），並對照[威脅模型](./threat-model.md)說明哪些對手會在意 Metadata、可以用哪些工具或習慣減少留下痕跡。
