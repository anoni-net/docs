# :material-shield-search: 威脅模型怎麼想

談匿名與隱私時，最常見的誤區是把工具當成答案，跳過釐清「我到底在防什麼」。威脅模型是一個結構化的思考方式，幫助你回答三個基本問題：要保護的資產是什麼，可能的對手是誰、他們有什麼能力，以及為了這份保護你願意付出多少時間、金錢與便利性的成本。這篇文章帶你跑一次威脅模型的建立流程，舉幾個常見角色（一般使用者、記者、社運參與者、家暴倖存者）的威脅模型範例做對照，搭配[匿名、隱私、假名、機密性的差別](./anonymity-vs-privacy.md)一起讀，下一步再進入[工具層](../tools/index.md)挑選對應的工具會更有方向。
