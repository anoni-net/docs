# :material-incognito-circle: 匿名、隱私、假名、機密性的差別

在日常對話裡，「匿名」「隱私」「假名」「機密」常常被混著用，但選工具、評估風險、設計流程時，把這四個詞拆開來看會差很多。匿名關注的是「沒人知道是你在做這件事」，隱私關注「我能控制誰看見我的個人資訊」，假名是「用一個固定身分活動，但這個身分和真實身分切開」，機密性則是「內容只有授權的人能讀」。這篇文章用具體場景對照這四個概念的界線，並說明選錯詞彙時容易誤判的工具與風險。
