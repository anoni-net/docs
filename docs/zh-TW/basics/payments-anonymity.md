# :material-cash-multiple: 為什麼匿名支付重要

談匿名與隱私時，金流常常被遺漏。一筆轉帳的時間、金額、收款人，配上信用卡號或銀行戶名，幾乎能還原一個人的社交網絡、活動範圍、價值傾向。這篇文章把支付當成 [Metadata](./metadata.md) 的一個獨立維度來看：先說明金流紀錄怎麼變成監控的一部分，再回頭看現金為什麼是最成熟的匿名支付，以及在跨境、線上、組織收受捐款的情境下現金的限制。讀完可接著看 [加密貨幣的隱私光譜](../tools/crypto-privacy-spectrum.md) 與 [倡議組織的匿名捐款管道](../scenarios/nonprofit-anonymous-donation.md)。
