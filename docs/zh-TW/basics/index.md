# :material-school-outline: 概念層

在挑選工具或學習操作之前，先把幾個關鍵概念釐清，後續閱讀工具層、場景層的內容會更順。這個分類從匿名網路社群在台灣關注的網路自由處境開始，再展開幾個常被混用的核心概念：「匿名」和「隱私」到底有什麼不同，威脅模型如何建立，Metadata 為什麼會洩漏比想像中更多的資訊，以及為什麼匿名支付是金流面向的關鍵拼圖。適合剛開始接觸匿名網路議題的讀者，先建立共同的動機與詞彙，再進入後面更具體的章節。
