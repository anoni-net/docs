# :material-shield-account-outline: 家暴受害者的數位準備

!!! warning "人身安全永遠優先"
    如果情況急迫，請先撥打 110 或 113 求助，再回頭看本頁。這頁不取代專業協助。

!!! info "撰寫中（2026 Q3）"
    這篇文章還在準備中，下方段落是主題大綱。歡迎到 [Matrix 社群](../community/tools.md) 表達想看的角度，或參與撰寫，見 [如何參與](../community/how-to-contribute.md)。社群追蹤連結 [issue #33](https://github.com/anoni-net/docs/issues/33)。

家暴情境下的數位風險，常常來自加害者擁有共用裝置、共用帳號、家庭計劃方案的存取權，或在被害者裝置上偷裝追蹤類 App。這篇文章從一個非線性的時間軸出發：在還沒離開時可以悄悄做的整理（不打草驚蛇），離開當下的優先順序（緊急聯絡、位置、金融），以及離開後重建獨立數位身分的步驟（新郵件、新號碼、新裝置、舊帳號的處置）。內容會引用衛福部、各地婦女救援基金會與相關支援組織提供的資源，並標註哪些步驟需要旁人協助。

要先理解風險為什麼存在，可以回頭看 [威脅模型怎麼想](../basics/threat-model.md) 與 [Metadata 是什麼](../basics/metadata.md)。緊急應對的整體清單見 [緊急求救](../help/index.md)。

## 將涵蓋的章節

完整版預計包含：

### 離開前的數位整理（不打草驚蛇）

- 一台加害者不知情的「對外裝置」：公共圖書館電腦、信任親友的裝置、預付卡手機的取得管道與成本
- 共用 Apple ID、Google 家庭方案、家庭 Wi-Fi 路由器紀錄的處置順序
- 搜尋紀錄、地圖紀錄、Google Activity 在共用裝置上的清理時機
- Find My、Google Family Link、共用日曆等定位類功能的關閉與替代
- 求助專線、安置機構聯繫方式的安全保存（不留在共用雲端）

### 離開過程的對外通訊

- 加害者在當事人裝置安裝追蹤類 App 的偵測與處理：手機端、車載 GPS、AirTag 與類似裝置
- 跟律師、社工、警察的聯繫管道選擇：信任親友的裝置、Signal 一次性號碼、預付卡的取得限制
- 銀行、保險、戶政等機構聯繫方式的更新順序，避免新地址被對方查到
- 113 保護專線、110 報案、地方法院家事庭緊急保護令的辦理流程速覽

### 離開後的數位重建

- 帳號密碼全面替換的優先順序：Email 與雲端最優先，再到社群、購物、串流
- 兩階段驗證從簡訊改為 TOTP（Authy、Aegis）或硬體金鑰
- 社群媒體切換到僅好友、檢視照片中的位置 metadata、過去合照的處理
- 共同好友的告知與篩選，社交工程詐騙的防範
- 跟蹤騷擾持續時的法律救濟：跟蹤騷擾防制法、保護令延長

## 在地脈絡

家暴情境的數位準備，要對齊台灣的在地資源與法律框架：

- **113 保護專線**：衛生福利部全國通報專線，匿名諮詢、社工轉介。手機與市話皆可撥
- **保護令申請**：透過警察、社政、地方法院家事庭。緊急保護令可在數小時內核發
- **跟蹤騷擾防制法**：2022 年施行，對不受歡迎的接近、跟監、騷擾電話、寄送物品有刑事追訴依據
- **現代婦女基金會**：[https://www.38.org.tw/](https://www.38.org.tw/){target="_blank"}（性別暴力諮詢與服務）
- **婦女救援基金會**：[https://www.twrf.org.tw/](https://www.twrf.org.tw/){target="_blank"}
- **勵馨基金會**：[https://www.goh.org.tw/](https://www.goh.org.tw/){target="_blank"}
- 各縣市政府社會局家庭暴力暨性侵害防治中心
