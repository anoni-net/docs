# :material-shield-account-outline: 家暴受害者的數位準備

家暴情境下的數位風險，常常來自加害者擁有共用裝置、共用帳號、家庭計劃方案的存取權，或在被害者裝置上偷裝追蹤類 App。這篇文章從一個非線性的時間軸出發：在還沒離開時可以悄悄做的整理（不打草驚蛇），離開當下的優先順序（緊急聯絡、位置、金融），以及離開後重建獨立數位身分的步驟（新郵件、新號碼、新裝置、舊帳號的處置）。內容會引用衛福部、各地婦女救援基金會與相關支援組織提供的資源，並標註哪些步驟需要旁人協助。
