# :material-bullhorn-outline: 社運行動者的數位準備

社運場景的數位風險和個人匿名不太一樣，往往涉及群組通訊、裝置被檢查、現場照片與位置資訊的暴露。這篇文章把一場行動拆成三個階段來談：動員前的帳號清理、社群通訊群組設定、訊息保存政策；行動中裝置的鎖定設定、即時通訊工具選擇、現場拍攝與上傳的注意事項；以及被臨檢、盤查、扣留時的數位應對。內容會搭配台灣與香港兩地行動圈常被討論的工具設定，並對照 [威脅模型](../basics/threat-model.md) 提供取捨。
