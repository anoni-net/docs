# :material-account-group-outline: 場景層

工具沒有絕對的好壞，重點是能不能對應到具體的處境與威脅。這個分類用幾個常見的角色與場景做說明：記者怎麼在和消息來源往返時降低洩漏風險、社運參與者在動員前後該做什麼準備、家暴倖存者在離開加害者前後如何整理數位足跡、LGBTQ+ 與性少數在不友善環境中的匿名社交策略，以及選舉觀察員的安全自保。每篇文章會搭配對應的工具與設定建議，可以反向回頭看 [概念層](../basics/index.md) 與 [工具層](../tools/index.md) 補上背景知識。
