# :material-hand-coin-outline: 倡議組織的匿名捐款管道

!!! info "撰寫中（2026 Q2–Q3）"
    這篇文章還在準備中，下方段落是主題大綱。歡迎到 [Matrix 社群](../community/tools.md) 表達想看的角度，或參與撰寫，見 [如何參與](../community/how-to-contribute.md)。

公民團體與倡議組織常常需要兩件看似衝突的事：一邊保護支持者、捐款人不被識別，一邊維持對外的財務透明與合規。這篇文章從兩個角色切入：組織方怎麼建立可選的匿名捐款管道（接受加密貨幣、現金、預付卡，搭配內部風險管理），捐款人怎麼選擇合適的工具與通道（避免讓自己暴露身分）。內容會引用 Tornado Cash 制裁案例作為提醒：合法、可稽核、保護弱勢三者要同時成立。對應 COSCUP 2026 的「公民團體與倡議組織」討論主軸，可搭配 [加密貨幣的隱私光譜](../tools/crypto-privacy-spectrum.md) 與 [台灣 VASP 法 2026](../taiwan/vasp-2026.md) 一起讀。
