!!! info "撰寫中（2026 Q3）"

對 LGBTQ+ 與性少數來說，是否、何時、向誰公開身分，往往不是法律問題而是日常處境問題：來自家庭、職場、學校或同儕的壓力，可能讓一個被動曝光的事件變成長期傷害。這篇文章談的是「如何保留連結，同時控管曝光節奏」：常見社交平台的隱私設定盲點、跨平台身分連動的洩漏路徑、相片與位置 Metadata 的處理、約會 App 與聊天工具在不同國家的風險差異，以及在台灣境外旅行時值得多留意的事項。對照 [匿名、隱私、假名、機密性的差別](../basics/anonymity-vs-privacy.md) 一起讀，更能理解這些做法背後保護的對象。
