# :material-gavel: 治理章程

社群運作到一定規模後，需要把「決定怎麼做」這件事寫下來，避免每次重新爭論基本原則。這份治理章程說明 anoni.net 社群的決策方式（共識決、表決、提案門檻）、成員角色（核心、貢獻者、觀察員）、爭議處理流程、行為準則與違反時的處置，以及章程本身的修訂方式。內容是一份活文件，重大調整會以社群提案流程進行討論。
