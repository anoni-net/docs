# :material-book-open-variant: 貢獻者百科

社群協作久了會累積許多「不成文規定」：標題該怎麼下、檔名怎麼命名、PR 描述要寫什麼、Issue 怎麼分類、新貢獻者第一週會碰到的問題。這篇貢獻者百科把這些散落在 README、Issue 留言、Discord 對話裡的內容整合成一頁，方便新成員一次看完，也讓資深成員有一個共同對話的依據。內容會持續更新，建議搭配 [如何參與與認領主題](./how-to-contribute.md) 與 [治理章程](./governance.md) 一起讀。
