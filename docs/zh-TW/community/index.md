# :material-account-group: 社群參與

我們以「**匿名網路**」為核心，透過 Tor、Tails、OONI 等工具在臺灣推廣網路自由與隱私。2026 年聚焦三大主題，歡迎有興趣的夥伴選擇一個主題加入討論與協作。我們使用自架 Matrix 家伺服器與申請制帳號，在隱私與社群品質之間取得平衡，背後考量可參見下方延伸閱讀。

!!! tip "第一次來，想先做點什麼？"

    很多事情不需要帳號就能開始：訂閱 [電子報](../contact.md) 跟讀社群動態、直接用 [社群自架服務](tools.md) 裡的 SearXNG 搜尋與 Send 傳檔、讀 [指南](../guides/index.md) 裡的隱私文章。想進一步參與討論，再來信申請 Matrix 帳號即可。

## 2026 三大主題

每個主題都有獨立的研究專題入口頁，整理研究目標、相關文章索引、進度與如何加入。

- [**個人隱私指引**](./privacy-guide.md)：整理可實際操作的隱私保護指引，依情境（日常、敏感工作、高風險）提供工具與步驟。
- [**Tor Relay 校園建立**](./relay-on-campus.md)：與 EFF、Tor Project 合作推動校園中繼節點，已有台師大成功案例，目標整理申請流程與活動企劃。
- [**匿名支付**](./payments-research.md)：探索現金以外情境下的匿名支付（穩定幣、區塊鏈應用），含法規與實作面向。

完整 2025 回顧與 2026 起步背景見：[延續 2025，走向 2026](../blog/posts/2025to2026.md)。
